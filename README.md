# TelecomX-Churn-Analysis
Análisis de evasión de clientes (Churn) para Telecom X: Pipeline de ETL desde API, limpieza de datos y diagnóstico de abandono con Python y Pandas.

Este repositorio contiene el desarrollo del Challenge Telecom X, donde asumí el rol de Analista de Datos para abordar un problema crítico de alta evasión de clientes. El proyecto abarca desde la extracción de datos mediante API hasta la entrega de un dataset procesado para modelos predictivos de Science de Datos.
